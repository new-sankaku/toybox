# Asset Leader（アセットリーダー）

## 概要

| 項目 | 内容 |
|-----|------|
| **役割** | アセット制作タスクの統括・管理 |
| **Phase** | Phase2: 開発 |
| **種別** | Leader |
| **入力** | イテレーション計画のasset_tasks |
| **出力** | 制作済みアセット群 |
| **Human確認** | アセット品質・スタイル統一を確認 |

---

## 責務

1. **タスク割り当て**
   - Asset Agentへのタスク分配
   - 優先度に基づく実行順序決定

2. **進捗管理**
   - 各Agentの進捗追跡
   - Code Leaderへの完成通知

3. **品質管理**
   - スタイルの一貫性確認
   - 技術仕様の適合確認

4. **Code連携**
   - Code Leaderからの要求受信
   - アセット納品通知の送信

---

## 配下のAsset Agents（動的生成）

| Agent | 担当 |
|-------|------|
| ImageAgent | 画像生成（DALL-E等） |
| AudioAgent | 音声生成（BGM/SE） |
| TextAgent | テキストコンテンツ |

※タスク内容に応じて動的に生成
